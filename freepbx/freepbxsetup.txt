You need to run sngrep from inside your FreePBX container to capture the SIP traffic between your PBX and your phone.

Here are the steps to do this:

1. Access Your Container: First, you need to get a command-line shell inside your running container. From your Windows command prompt, run:

bash
docker exec -it freepbx-app bash
2. Install sngrep: The tiredofit/frepbx image you're using might not have sngrep pre-installed. To install it, run these two commands inside the container's shell:

bash
apt update
apt install -y sngrep
3. Run sngrep: Once the installation is complete, simply type:

bash
sngrep
4. Capture SIP Traffic: With sngrep now running, place a call between your extensions. The tool will automatically display the SIP message flow, allowing you to inspect the c= lines in the SDP to verify the IP addresses being used for the audio stream (RTP).

Once you've finished capturing, you can type quit to exit sngrep. If you need to exit the container's shell, just type exit.
